# GrokChat Mobile (Expo)

Минимальный мобильный UI для вашего Rust gateway.

Текущая версия: Expo SDK 54.

## Что умеет

- подключаться к `ws://<server-ip>:8080/ws`
- показывать входящие сообщения
- отправлять сообщения в p2p-чат

## 1) Запусти Rust backend

В корне проекта:

1. `cargo run`
2. Убедись, что в консоли есть строка про gateway:
   `UI gateway запущен на http://0.0.0.0:8080`

## 2) Запусти mobile UI

В папке `mobile`:

1. `npm install`
2. `npx expo start`
3. Открой через Expo Go на телефоне (QR-код)

Если в PowerShell ошибка про `npx`/ExecutionPolicy, используй:

- `npx.cmd expo start`

## 3) Подключение в приложении

- `IP сервера` = IP машины, где запущен Rust (например `192.168.1.10`)
- `Порт` = `8080`
- Нажми `Подключить`

## Формат WS

Приложение отправляет:

```json
{"type":"send_message","text":"привет"}
```

Backend присылает:

```json
{"type":"chat_message","from":"<peer_id>","text":"..."}
```

## Важно

Телефон и ПК должны быть в одной локальной сети.
Если не коннектится, проверь firewall на порту `8080`.
